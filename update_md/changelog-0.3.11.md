# 更新日志
## fileMapping 0.3.11


# 对比上次更新了什么？

- `__end__` 可以定义当程序结束时执行的函数，可以用来做一些清理工作。


# 如使用？
```python
# __init__.py


def end():
    print('程序结束')

__end__ = 'end'
# __end__ = end # 也可以直接传入函数


```
